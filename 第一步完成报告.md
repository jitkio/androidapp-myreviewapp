## 📝 使用示例

### 使用间距
```kotlin
// 旧代码
padding(horizontal = 20.dp)  // ❌ 不符合规范

// 新代码
padding(horizontal = Spacing.pageHorizontal)  // ✅ 24dp
padding(Spacing.space4)  // ✅ 16dp
```

### 使用颜色
```kotlin
// 旧代码
color = MaterialTheme.colorScheme.primary  // 可能是紫色

// 新代码
color = Primary  // ✅ 蓝色 #4F6EF7
backgroundColor = BgCard  // ✅ 卡片背景 #1A1D27
```

### 使用字体
```kotlin
// 标题
Text(
    text = "Today",
    style = MaterialTheme.typography.titleLarge  // T2 20sp Bold
)

// 正文
Text(
    text = "内容",
    style = MaterialTheme.typography.bodyLarge  // T3 16sp Regular
)

// 辅助文字
Text(
    text = "12:30",
    style = MaterialTheme.typography.bodySmall  // T4 13sp Regular
)
```

### 使用圆角
```kotlin
// 卡片
Card(
    shape = RoundedCornerShape(CornerRadius.card)  // 16dp
)

// 按钮
Button(
    shape = RoundedCornerShape(CornerRadius.medium)  // 14dp
)
```

### 使用动画
```kotlin
// 点击反馈
val scale by animateFloatAsState(
    targetValue = if (pressed) AnimationParams.buttonPressScale else 1f,
    animationSpec = AnimationSpec.clickFeedback
)
```
