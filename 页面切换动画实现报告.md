## 🎯 动画规则

### 底部导航栏切换

**规则：** 根据Tab索引比较决定滑动方向

| 当前页面 | 目标页面 | 动画效果 | 说明 |
|---------|---------|---------|------|
| Library (1) | Today (0) | 向右滑 | 0 < 1，去左边的Tab |
| Library (1) | Map (2) | 向左滑 | 2 > 1，去右边的Tab |
| Today (0) | Me (4) | 向左滑 | 4 > 0，去右边的Tab |
| Me (4) | Today (0) | 向右滑 | 0 < 4，去左边的Tab |

**动画参数：**
- 时长：280ms（使用 `AnimationDuration.pageTransition`）
- 缓动函数：tween（ease-out效果）

### 详情页导航

**规则：** 前进向左滑，后退向右滑

| 导航类型 | 动画效果 | 说明 |
|---------|---------|------|
| 主页 → 详情页 | 从右向左滑入 | 前进导航 |
| 详情页 → 主页 | 从左向右滑出 | 后退导航 |
| 详情页 → 详情页 | 从右向左滑入 | 深层导航 |

**动画参数：**
- 时长：280ms
- 缓动函数：tween

### 核心实现

#### 1. Tab路由映射

```kotlin
// Map routes to tab indices
val tabRoutes = listOf(
    Screen.Today.route,      // 0
    Screen.Library.route,    // 1
    Screen.Map.route,        // 2
    Screen.Practice.route,   // 3
    Screen.Me.route          // 4
)
```

#### 2. 进入动画（enterTransition）

```kotlin
enterTransition = {
    val targetRoute = targetState.destination.route
    val initialRoute = initialState.destination.route

    val targetIndex = tabRoutes.indexOf(targetRoute)
    val initialIndex = tabRoutes.indexOf(initialRoute)

    if (targetIndex != -1 && initialIndex != -1) {
        // 底部Tab导航 - 根据相对位置滑动
        if (targetIndex > initialIndex) {
            // 去右边的Tab - 从右滑入
            slideInHorizontally(
                initialOffsetX = { it },
                animationSpec = tween(AnimationDuration.pageTransition)
            )
        } else {
            // 去左边的Tab - 从左滑入
            slideInHorizontally(
                initialOffsetX = { -it },
                animationSpec = tween(AnimationDuration.pageTransition)
            )
        }
    } else if (targetIndex == -1) {
        // 去详情页 - 从右滑入
        slideInHorizontally(
            initialOffsetX = { it },
            animationSpec = tween(AnimationDuration.pageTransition)
        )
    } else {
        // 返回Tab - 从左滑入
        slideInHorizontally(
            initialOffsetX = { -it },
            animationSpec = tween(AnimationDuration.pageTransition)
        )
    }
}
```

#### 3. 退出动画（exitTransition）

```kotlin
exitTransition = {
    val targetRoute = targetState.destination.route
    val initialRoute = initialState.destination.route

    val targetIndex = tabRoutes.indexOf(targetRoute)
    val initialIndex = tabRoutes.indexOf(initialRoute)

    if (targetIndex != -1 && initialIndex != -1) {
        // 底部Tab导航
        if (targetIndex > initialIndex) {
            // 去右边的Tab - 当前页向左滑出
            slideOutHorizontally(
                targetOffsetX = { -it },
                animationSpec = tween(AnimationDuration.pageTransition)
            )
        } else {
            // 去左边的Tab - 当前页向右滑出
            slideOutHorizontally(
                targetOffsetX = { it },
                animationSpec = tween(AnimationDuration.pageTransition)
            )
        }
    } else if (targetIndex == -1) {
        // 去详情页 - 当前页向左滑出
        slideOutHorizontally(
            targetOffsetX = { -it },
            animationSpec = tween(AnimationDuration.pageTransition)
        )
    } else {
        // 返回Tab - 当前页向右滑出
        slideOutHorizontally(
            targetOffsetX = { it },
            animationSpec = tween(AnimationDuration.pageTransition)
        )
    }
}
```

## 📝 注意事项

### 性能优化

1. **动画时长适中**
   - 280ms不会造成卡顿感
   - 不会影响快速切换

2. **硬件加速**
   - Compose自动使用硬件加速
   - 滑动动画性能优秀

3. **内存管理**
   - 动画结束后自动释放资源
   - 不会造成内存泄漏

### 开发建议

1. **保持一致性**
   - 所有新增页面自动应用动画
   - 无需额外配置

2. **避免冲突**
   - 不要在单个页面内部使用相同的滑动动画
